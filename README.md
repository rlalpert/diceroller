# Diceroller

* Rolls DnD style dice.

# To-Do

# Setup Virtual Environment

`python3 -m venv env`

`source env/bin/activate`

# Install Packages

`pip install -r requirements.txt`

# Use

`python diceroller.py`

# Testing

To watch (run from project directory):
`pytest -f .`

Detailed test info:
`pytest -v`
